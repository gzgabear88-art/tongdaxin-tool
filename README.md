# 通达信自动下载工具 v1.0.0

## 这是什么？

这是一个**全自动下载通达信股票数据**的工具。

## 功能

- ✅ 每天自动下载股票数据（12:05 和 15:05）
- ✅ 全自动鼠标点击和键盘操作
- ✅ 下载完成后发送短信通知
- ✅ 操作日志记录
- ✅ 一键安装，傻瓜式操作

## 使用方法

### 1. 下载 exe 文件

点击上方 **Releases** 或 **Actions** 页面，下载最新的 `.exe` 文件

### 2. 配置通达信路径

用记事本打开 `config.ini`，修改这一行：
```ini
tongdaxin_path = C:\你的通达信安装路径
```
改成你电脑上通达信的实际路径，例如：
```ini
tongdaxin_path = C:\同花顺\TdxW.exe
```

### 3. 运行程序

双击 `通达信自动下载工具_v1.0.0.exe`，程序会自动：
1. 启动通达信
2. 登录
3. 下载日线数据和实时行情
4. 导出Excel文件
5. 关闭通达信
6. 发送短信通知

## 常见问题

### Q: 为什么exe会闪退？
A: 可能是config.ini里的通达信路径不对，确保路径指向包含TdxW.exe或TongDaXin.exe的文件夹

### Q: 点击位置不对怎么办？
A: 不同屏幕分辨率点击位置会不同，查看logs文件夹里的日志可以调整

### Q: 怎么修改下载时间？
A: 打开config.ini，修改morning_time和afternoon_time，例如改成13:00和16:00
