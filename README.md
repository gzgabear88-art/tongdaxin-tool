# 通达信自动下载工具

## 📋 项目简介

本工具用于自动化操作通达信金融终端软件，帮助用户自动完成以下工作流程：

1. 🔓 打开通达信软件
2. 👤 自动登录账号
3. ⌨️ 输入板块代码（51）
4. 📥 下载盘后数据（日线+实时行情）
5. 📊 导出数据到Excel文件
6. 📱 通知用户任务完成

## ✨ 主要特性

- **图形界面** - 简洁易用的可视化界面
- **图像识别** - 使用OpenCV定位按钮，兼容不同版本通达信
- **安全验证** - 每步操作前验证目标存在，找不到即报错
- **定时执行** - 支持每天12:35和15:35自动执行
- **即时终止** - 随时可以停止正在运行的任务
- **微信通知** - 任务完成后通过微信消息通知用户

## 🛠️ 技术架构

| 模块 | 说明 |
|------|------|
| main.py | 主程序入口，GUI界面 |
| config_manager.py | 配置管理，账号加密存储 |
| auto_operation.py | 自动化操作引擎 |
| notifier.py | 通知发送模块 |
| scheduler.py | 定时任务调度器 |

## 📦 安装与运行

### 环境要求

- Windows 10 或更高版本
- Python 3.8+
- 通达信金融终端软件

### 安装步骤

1. **克隆或下载本项目**

2. **安装Python依赖**
```bash
pip install -r requirements.txt
```

3. **配置通达信路径**
- 首次运行会提示选择通达信软件路径
- 或手动在界面中设置

4. **运行程序**
```bash
python main.py
```

### 生成Windows可执行文件

```bash
pyinstaller --onefile --windowed main.py
```

生成的可执行文件在 `dist/main.exe`

## 🎯 使用说明

### 首次使用

1. 打开程序
2. 输入通达信账号和密码（可选择保存）
3. 选择通达信软件安装路径
4. 选择数据保存路径
5. 点击"开始执行"

### 执行模式

- **立即执行** - 立即开始自动化操作
- **定时执行** - 按设定时间自动运行（12:35 / 15:35）
- **终止任务** - 停止正在运行的任务

## 🔒 安全机制

程序采用多重安全验证：

1. **操作前验证** - 每步操作前先识别目标元素
2. **失败即停** - 找不到目标立即报错，不盲目继续
3. **用户可控** - 随时可终止任务
4. **配置加密** - 账号密码本地加密存储

## ⚠️ 注意事项

1. **图像识别** - 首次使用可能需要为您的通达信版本截图
2. **分辨率** - 不同屏幕分辨率可能需要调整
3. **通达信版本** - 不同版本界面可能有差异
